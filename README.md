# Emotion-prediction-
🧠 Emotion Prediction App
An NLP-based web application that predicts human emotions from text input using Machine Learning techniques. This project is built using Python, Scikit-learn, and deployed with Streamlit.

🚀 Features
•	Predicts emotion from user-entered text
•	Uses TF-IDF for text vectorization
•	Machine Learning model (Logistic Regression) for classification
•	Real-time predictions using Streamlit interface

🛠️ Tech Stack
•	Python
•	Pandas, NumPy
•	Scikit-learn
•	Streamlit

📂 Project Structure
emotion-prediction/
│── app.py
│── requirements.txt
│── Emotion Phrases.csv
│── README.md

▶️ How to Run Locally
1.	Clone the repository
git clone https://github.com/your-username/emotion-prediction.git
cd emotion-prediction
2.	Install dependencies
pip install -r requirements.txt
3.	Run the app
streamlit run app.py

📊 How It Works
•	Text input is cleaned and preprocessed
•	TF-IDF vectorization converts text into numerical features
•	Machine learning model predicts the emotion
•	Output is displayed instantly on the UI
